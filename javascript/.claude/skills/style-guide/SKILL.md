# Michelangelo JavaScript/TypeScript Style Guide

Entry point for coding standards. Claude loads the relevant sub-skill automatically based on what you're working on.

## Sub-Skills

| Working on | Skill |
|---|---|
| Any code | `principles` |
| React components | `react` |
| Tests | `tests` |
| TypeScript types | `typescript` |
| File/directory structure or imports | `files` |
| Comments, JSDoc, inline documentation | `docs` |

## Core Principle

**Follow existing patterns in the codebase first.** Look for similar functionality already present, use the same approach and structure, and establish new patterns carefully when none exist.
