# Michelangelo Release Notes

## 2025-08-06 to 2025-08-20

### 🚀 Major Features

#### **Enhanced Table System** `[Frontend]`
Tables now include pagination, column controls, sorting, and drag-and-drop column reordering with state persistence across sessions. _(792cb499, ed48c16d, 12e5cde7, a5cc3c46, 13664c14)_

#### **Configuration-Driven Project Views** `[Frontend]`
Replaced hard-coded pipeline and run components with configurable PhaseEntityView system, enabling new entity types through configuration rather than code changes. _(eeefda93, c77d1b1b, 3a2f9429, bdbbb0f5)_

#### **Improved Execution Views** `[Frontend]`
Added schema-based task rendering, recursive subtask navigation, and click-to-scroll between overview and details. _(6747b158, 4b552778, 394dc4d4, 0c6fb560)_

### 🏗️ Infrastructure & Platform

#### **Cluster Management** `[Backend Services]`
New Cluster CRD and service with Kubernetes proto messages for cluster lifecycle management. _(c59a92d7)_

#### **API Extensibility** `[Backend Services]` 
Added `ext` fields to all Spec and Status messages for future extensibility without breaking changes. _(6f861cbc)_

#### **Worker System** `[Backend Services]`
Refactored worker plugins for improved modularity and added validation hook system. _(1fcccfee, 213b94db)_

### 🛠️ Developer Experience

#### **Python SDK & CLI** `[SDK/Tooling]`
- Automated release pipeline with GitHub Actions and dynamic versioning _(b2271aa3, 532debde)_
- Enhanced CLI with JSON output formatting _(43e05119)_ 
- Improved uniflow pipeline registration _(265b7331)_

#### **Development Tooling** `[Infrastructure]`
- Go IDE setup with gopackagesdriver for VSCode/Cursor _(c1971ba9)_
- Sandbox support for separate Ray jobs clusters _(9498f035)_
- Ray Manager service account with RBAC for Ray jobs _(b744abb6)_

---

**Breaking Changes**: None

**Migration Notes**: All changes maintain backward compatibility. Table and execution view improvements are automatically available to existing components.

**Components Updated**: Frontend, Backend Services, Python SDK, CLI Tools, Proto APIs
